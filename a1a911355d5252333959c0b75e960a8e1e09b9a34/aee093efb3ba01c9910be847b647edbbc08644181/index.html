Here we calculate three biodiversity components from deployed ecoacoustic devices across various farms in Thailand — Taxonomic Diversity (TD), Functional Diversity (FD), and Structural Diversity (SD) — and combines them into the Multidimensional Biodiversity Index (MBI). Bootstrap simulations are used to quantify uncertainty around each index.

Integrating Biodiversity Indexes:

Biodiversity indexes are mathematical measures used by ecologists to quantify the variety of life in an ecosystem.

Here we develop a new approach to calculate three biodiversity indexes that will be combined as the Multidimensional Biodiversity Index (MBI) at the end.

Taxonomic Diversity (TD)

This is not just species richness. Here we calculated the TD as the geometric mean of occupancy for each species, representing the abundance of each species at the sampled places.

\[ TD_{t}=exp(\frac{1}{n}\sum_{i}^{} ln \frac{d_{i,t}}{d_{i,1}}) \] were:
- n = number of species.
- di,t = abundance of species i at time t.
- di,1 = abundance of species i at time 1.

Since we have just one temporal measurement the base line to compare is TD at Monoculture.

The occupancy for each species was calculated using the package spOccupancy resulting as a table of mean occupancy of posteriors per site.

Calculating TD per grouping factor

The TD formula is the geometric mean of occupancy ratios relative to the baseline period \(\hat{\psi}_1\). Using model-estimated \(\hat{\psi}_t\)

\[TD_t = \exp\!\left(\frac{1}{n}\sum_{i=1}^{n} \ln\frac{\hat{\psi}_{i,t}}{\hat{\psi}_{i,1}}\right)\]

Code
#' @param psi_t  Numeric vector: estimated occupancy at time t  (length n_sp)
#' @param psi_1  Numeric vector: estimated occupancy at baseline (length n_sp)
#' @return Scalar TD value (1 = no community-level change from baseline)

calc_TD <- function(psi_t, psi_1) {
  psi_t[psi_t <= 0] <- 1e-6
  psi_1[psi_1 <= 0] <- 1e-6
  exp(mean(log(psi_t / psi_1)))
}

Functional Diversity (FD)

NoteFunctional Diversity (FD):

Should be calculated as Functional Divergence using the package fundiversity and the function fd_fdiv, since input values are continuous occupancy probabilities (ψ, 0–1), not binary presence absence.

Functional Divergence represents how abundance (occupancy) is spread along a functional trait axis, within the range occupied by the community. For instance, divergence is low when the most abundant species have functional traits that are close to the center of the functional trait range. Conversely, when the most abundant species have extreme functional trait values, then divergence is high. In a multivariate context, functional divergence relates to how abundance is distributed within the volume of functional trait space occupied by species.

Occupancy was already estimated using the package spOccupancy, and is comming from the posteriors located at a three dimensional array iter_sp_site where each species was detected in each site, corrected by detectability, using 3000 iterations. The psi values in the array iter_sp_site have 3000 iterations per species per site in that order. The psi values of the posteriors are used in the Functional Diversity formula and are provided by array psi_raw_THAILAND_VF.rds.

Calculating Functional Diversity requires species-level trait data. Traits for each species are provided by the table avonet_traits_birdlife.csv located at the data folder.

Functional Diversity uses four traits:

  • Wing Length.
  • Mass.
  • Habitat Density.
  • Trophic Level.

Functional Diversity comes without error or confidence intervals. So we need to implement a bootstrap and jackknife strategy to estimate error per site and per grouping factor using the posteriors (calculated using the spOccupancy package) as the re-sampling basis for psi. The grouping factors per site are located in the column integration_class of the table psi_mean_THAILAND_VF.csv saved at the data folder.

#' Load and align trait and occupancy data
load_and_clean_data <- function() {
  # Use relative paths
  # Load occupancy, traits, and FHD stats
  site_comm_raw <- read_csv("C:/CodigoR/WildMon/FunctionalDiversityV3/data/psi_mean_THAILAND_VF.csv", show_col_types = FALSE) %>%
    rename(site_name = `site name`) %>%
    mutate(site_name = str_replace_all(site_name, " ", "_"))

  iter_sp_site <- readRDS("C:/CodigoR/WildMon/FunctionalDiversityV3/data/psi_raw_THAILAND_VF.rds")
  traits_raw <- read_csv("C:/CodigoR/WildMon/FunctionalDiversityV3/data/avonet_traits_birdlife.csv", show_col_types = FALSE)


  # Align species
  common_species <- intersect(colnames(site_comm_raw), traits_raw$Species1)


  # Select and format traits
  traits <- traits_raw %>%
    filter(Species1 %in% common_species) %>%
    select(Species1, Wing.Length, Mass, Habitat.Density, Trophic.Level) %>%
    mutate(across(c(Wing.Length, Mass, Habitat.Density, Trophic.Level), as.numeric)) %>%
    mutate(across(c(Wing.Length, Mass, Habitat.Density, Trophic.Level), ~ if_else(.x == 0, NA_real_, .x))) %>%
    mutate(across(c(Wing.Length, Mass, Habitat.Density, Trophic.Level), log)) %>%
    drop_na() %>%
    column_to_rownames("Species1")

  common_species <- rownames(traits)

  site_comm <- site_comm_raw %>%
    select(site_name, integration_class, all_of(common_species))

  # Transpose the array to have: species, sites, and iteration
  # The RDS is (iteration, species, site)
  sp_site_iter <- aperm(iter_sp_site, c(2, 3, 1))

  # Ensure dim names match
  dimnames(sp_site_iter) <- list(
    species = dimnames(iter_sp_site)[[2]],
    sites = dimnames(iter_sp_site)[[3]],
    iteration = NULL
  )

  return(list(traits = traits, site_comm = site_comm, sp_site_iter = sp_site_iter))
}


#' Estimate site-level uncertainty using posterior iterations and species jackknife
estimate_site_uncertainty <- function(obs_fdiv_df, traits, site_comm, sp_site_iter, common_species) {
  site_results <- obs_fdiv_df %>%
    left_join(site_comm %>% select(site_name, integration_class), by = "site_name") %>%
    mutate(
      post_mean = NA_real_,
      post_low = NA_real_,
      post_high = NA_real_,
      jack_low = NA_real_,
      jack_high = NA_real_
    )

  sites <- site_results$site_name
  all_posterior_sites <- dimnames(sp_site_iter)$sites

  for (i in seq_along(sites)) {
    s_name <- sites[i]

    if (!(s_name %in% all_posterior_sites)) next

    # 1. Posterior Uncertainty
    # Sample iterations
    iter_indices <- sample(1:dim(sp_site_iter)[3], N_ITER)
    # Subset psi for this site across sampled iterations
    site_psi_samples <- sp_site_iter[common_species, s_name, iter_indices] # species x iterations

    post_fdiv_vals <- apply(site_psi_samples, 2, function(psi_vec) {
      comm_mat <- matrix(psi_vec, nrow = 1)
      colnames(comm_mat) <- common_species
      rownames(comm_mat) <- "site"
      res <- tryCatch(fd_fdiv(traits, comm_mat), error = function(e) NULL)
      if (is.null(res)) {
        return(NA)
      }
      return(res$FDiv)
    })

    site_results$post_mean[i] <- mean(post_fdiv_vals, na.rm = TRUE)
    site_results$post_low[i] <- quantile(post_fdiv_vals, (1 - CONF_LEVEL) / 2, na.rm = TRUE)
    site_results$post_high[i] <- quantile(post_fdiv_vals, 1 - (1 - CONF_LEVEL) / 2, na.rm = TRUE)

    # 2. Species Jackknife (using mean psi)
    site_psi_mean <- site_comm %>%
      filter(site_name == s_name) %>%
      select(all_of(common_species)) %>%
      as.numeric()
    names(site_psi_mean) <- common_species

    jack_vals <- sapply(seq_along(common_species), function(idx) {
      sp_subset <- common_species[-idx]
      traits_subset <- traits[sp_subset, , drop = FALSE]
      comm_mat <- matrix(site_psi_mean[sp_subset], nrow = 1)
      colnames(comm_mat) <- sp_subset
      rownames(comm_mat) <- "site"

      res <- tryCatch(fd_fdiv(traits_subset, comm_mat), error = function(e) NULL)
      if (is.null(res)) {
        return(NA)
      }
      return(res$FDiv)
    })

    # Standard Jackknife CI
    obs_val <- site_results$FDiv_obs[i]
    n_sp <- length(common_species)
    jack_mean <- mean(jack_vals, na.rm = TRUE)
    jack_se <- sqrt(((n_sp - 1) / n_sp) * sum((jack_vals - jack_mean)^2, na.rm = TRUE))

    t_crit <- qt(1 - (1 - CONF_LEVEL) / 2, df = n_sp - 1)
    site_results$jack_low[i] <- obs_val - t_crit * jack_se
    site_results$jack_high[i] <- obs_val + t_crit * jack_se

    if (i %% 5 == 0) message(paste("Processed", i, "/", nrow(site_results), "sites"))
  }

  return(site_results)
}


#' Calculate group-level summaries and confidence intervals via bootstrapping
calculate_group_summary <- function(site_results) {
  group_summary <- site_results %>%
    group_by(integration_class) %>%
    summarise(
      mean_FDiv_obs = mean(FDiv_obs, na.rm = TRUE),
      n_sites = n(),
      .groups = "drop"
    )

  group_boot_stats <- site_results %>%
    filter(!is.na(FDiv_obs)) %>%
    group_by(integration_class) %>%
    do({
      data <- .$FDiv_obs
      if (length(data) > 1) {
        b_obj <- boot(data = data, statistic = function(d, i) mean(d[i], na.rm = TRUE), R = 500)
        b_ci <- tryCatch(boot.ci(b_obj, type = "perc", conf = CONF_LEVEL), error = function(e) NULL)
        if (!is.null(b_ci)) {
          data.frame(
            group_boot_low = b_ci$percent[4],
            group_boot_high = b_ci$percent[5]
          )
        } else {
          data.frame(group_boot_low = NA_real_, group_boot_high = NA_real_)
        }
      } else {
        data.frame(group_boot_low = NA_real_, group_boot_high = NA_real_)
      }
    }) %>%
    ungroup()

  group_summary <- group_summary %>%
    left_join(group_boot_stats, by = "integration_class")

  return(group_summary)
}

Structural Diversity (SD)

Shannon entropy of the foliage height class distribution, normalized by the baseline entropy so the index equals 1 at \(t = 1\).

\[SD_t = \frac{-\sum_{l=1}^{L} f_{l,t}\ln f_{l,t}}{SD_1}\]

#' @param foliage_t  Proportions per height layer at time t
#' @param foliage_1  Proportions at baseline
#' @return Scalar SD value (1 at baseline)

# function to calculate SD
calc_SD <- function(foliage_t, foliage_1) {
  H <- function(p) { p <- p[p > 0]; -sum(p * log(p)) }
  H(foliage_t) / H(foliage_1)
}
